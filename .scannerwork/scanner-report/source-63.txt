    <h1>User Stories</h1>

    <ol type="1">
        <li>As a <a href="Atlante's Dictionary/Dictionary entries/textbookSeller.html">textbook seller</a>, I want to sell my old books, so that I can earn money from something I don't need anymore.</li>
        <li>As a Tutor, I want to submit my <a href="Atlante's Dictionary/Dictionary entries/tutoringData.html">tutoring data</a>, so that students in need can contact me.</li>
        <li>As a student, I want a calendar in which I can add events about my study sessions,  so that I can keep track of how much I spend studying.</li>
        <li>As a student, I want to see a tutor’s received reviews, so that I can choose the best tutor.</li>
        <li>As a <a href="Atlante's Dictionary/Dictionary entries/textbookSeller.html">textbook seller</a>, I want to show images of the textbooks I want to sell, so that I can be trusted by the buyers.</li>
        <li>As a <a href="Atlante's Dictionary/Dictionary entries/textbookSeller.html">textbook seller</a>, I want to show that the prices of the textbooks I want to sell can be negotiable, so that I can sell the textbooks faster.</li>
        <li>As a student, I want to receive plots of the frequency of my activities, so that I can monitor my progress and improve accordingly.</li>
        <li>As a student, I want to import my study plan, so that I can easily check how many exams I have to take before graduation.</li>
        <li>As a student, I want to record the grades of every exam I take and see my average, so that I can check my overall performance.</li>
        <li>As a student, I want to Import my lessons’ schedule into a calendar, so that I can receive reminders before the start of a lesson.</li>
        <li>As a student, I want to leave a report to a scam textbook seller, so that I can help other students avoid frauds.</li>
        <li>As a <a href="Atlante's Dictionary/Dictionary entries/textbookSeller.html">textbook seller</a>, I want to submit my contact numbers, so that customers can contact me.</li>
    </ol>
